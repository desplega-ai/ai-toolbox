This project serves as a list of sub-projects and tools that can be used for different purposes. 

## Usage

1. List folders to get a general idea of the available projects.
2. Check READMEs in the applicable folders
3. cd into the folder of interest
4. Check toolchain used to run the porject (generally either node, bash, bun or python via pyproject.toml)
5. Proceed with your goal

## Development

1. Create a new folder with a minimal slug (manually or via cli toolchain generator)
2. Add a README.md with a description of the project
3. Proceed with development based on your goal
4. Keep README minimal, but clear of the purpose of the project
